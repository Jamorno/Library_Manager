# Library Manager (CLI Edition)

A command-line interface (CLI) application to manage a library system. You can add books, borrow/return them, view reports, and save data to a JSON file.

## Features

- Add books with stock quantity
- Borrow and return books
- View available stock and borrowed books
- Automatically saves and loads data to/from `library.json`

## How to Run

1. Make sure you have Python 3 installed.
2. Run the main script:

```bash
python main.py
